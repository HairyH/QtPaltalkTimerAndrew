Description: 
QtPaltalkTimerAndrew.exe is a simple, easy to install and use. mic timer, for the new, Paltalk client 
application. It periodically sends text messages, at adjustable time intervals, to a Paltalk chatroom, 
indicating, that the current mic user how much time is on mic. It also sends the text in bold, time 
limit and auto dots the mic user off the mic, when the corresponding check boxes selected.
